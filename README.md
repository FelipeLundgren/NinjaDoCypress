NinjaDoCypress
Projeto de estudos focado em automação de testes com Cypress, abrangendo desde os fundamentos de E2E até integrações avançadas de API e nuvem. Este repositório reflete minha preparação técnica como QA, focada em boas práticas e qualidade de software.

🚀 Tecnologias
Framework: Cypress

Linguagem: JavaScript/TypeScript

Integração: Cypress Cloud & CI/CD

🛠️ Jornada de Aprendizado
O projeto documenta a evolução da minha prática em automação:

Fundamentos E2E: Configuração, seleção de elementos resilientes (data-cy) e interações com a interface.

Testes de API: Validação de endpoints, status codes e payloads utilizando cy.request().

Estruturação: Uso de Custom Commands e Page Objects para manter um código limpo e escalável.

Cloud & CI: Configuração para execução headless, integração com Cypress Cloud para gestão de evidências (vídeos/screenshots) e automação em pipelines.

📋 Como Executar
Clone o repositório: git clone [https://github.com/FelipeLundgren/NinjaDoCypress.git](https://github.com/FelipeLundgren/NinjaDoCypress.git)

Instale as dependências: npm install

Execute os testes:

Modo interativo: npx cypress open

Modo headless: npx cypress run

💡 Objetivo
Este projeto integra minha trilha de estudos para a certificação CTFL (ISTQB), aplicando conceitos reais de qualidade desde a camada de API até a interface do usuário.
